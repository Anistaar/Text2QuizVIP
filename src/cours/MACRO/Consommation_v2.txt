### Formules (à retenir)
— Keynes (CT)
• C = C0 + c·Y ; S = Y − C
• PMC = C/Y ; PME = 1 − PMC ; PmC = c ; PmE = 1 − c
• k = 1/(1−c) ; Y* = k·(C0 + I0)

— Avec impôts forfaitaires
• Yd = Y − T ; C = C0 + c·(Y − T)
• Y* = [C0 − cT + I0] / (1 − c) ;  k_T = −c/(1−c)

— Modigliani
• C_t = α·Y^L_t + β·A_t ; S/Y ≈ g·(A/Y) si A/Y ≈ constant

— Friedman
• Y = Yp + Yt ; C = γ·Yp ;  Yp ≈ (1−λ)·(Y_t + λY_{t−1} + …)
